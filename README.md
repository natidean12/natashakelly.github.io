# 🌱 EcoSapiens

Site institucional desenvolvido para a EcoSapiens, empresa especializada em engenharia, arquitetura ecológica, permacultura e soluções sustentáveis.

## 📖 Sobre o projeto

O objetivo deste projeto é apresentar os serviços, metodologia e tecnologias da EcoSapiens por meio de um site moderno, responsivo e intuitivo.

O desenvolvimento priorizou uma identidade visual inspirada na natureza, utilizando tons de verde e terra, animações suaves e uma navegação simples para proporcionar uma boa experiência ao usuário.

## ✨ Funcionalidades

- Página inicial moderna
- Navegação por seções
- Layout totalmente responsivo
- Cards informativos
- Área de projetos
- Formulário de contato
- Botão flutuante do WhatsApp
- Interface inspirada em sustentabilidade e design ecológico

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## 🎯 Objetivo

Desenvolver uma interface profissional para representar a identidade da EcoSapiens, valorizando conceitos de sustentabilidade, bioconstrução e inovação ambiental.

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em computadores, tablets e smartphones.
